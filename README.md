- **Regras de Negócio e Casos Extremos (Edge Cases):** Criei uma lógica dinâmica para gestão de horários no `script.js`. A aplicação diferencia horários normais (8h às 18h) dos horários de sábado (8h às 13h) e bloqueia completamente agendamentos aos domingos com notificação ao usuário. Além disso, incluí tratamento de fuso horário (Timezone parsing bug) para evitar falhas de leitura de data e um sistema defensivo que desabilita horas do passado (ex: o usuário não consegue agendar algo para as 08:00h do dia atual se a requisição for feita às 14:00h).
